# PyESPCar色块追踪



## 简介

![pyespcar tracker](./image/小车色块追踪.png)

使用PyESPCar还有OpenCV实现色块追踪, OpenCV识别色块,然后计算偏移量,通过MQTT协议发送给PyESPCar小车. 源码见:https://github.com/1zlab/1ZLAB_PyEspCar/tree/master/examples/02-color-block-track
